{action} d'un post

Insérer un lien :
  1. vers un site ou un document en ligne :
    • cliquez sur
    • remplissez le champ "URL" avec le lien vers la page
    • remplissez le champ "texte" avec texte sur lequel il faudra cliquer
    • validez
  2. vers un document sur votre ordinateur :
    • cliquez sur "choisir un fichier" (sous la zone de texte)
    • sélectionnez le document voulu
    • cliquez sur "Ajouter"
    • copiez le lien qui s'affiche sous la zone de téléchargement (par exemple : docs/nom_du_fichier.pdf)
    • suivez les étapes du paragraphe 1 en collant le lien dans le champ "URL"
Choisissez ensuite la section dans laquelle ranger le post et cochez les options si besoin.

Validez, c'est fait !
